# SpaceX-Insight-Data

##📝 Présentation du Projet
Ce projet final (Capstone) analyse les données de lancement de SpaceX pour prédire le succès de l'atterrissage du premier étage des fusées Falcon 9. La capacité de réutiliser ces boosters est le facteur clé qui permet à SpaceX de réduire drastiquement les coûts de mise en orbite.

Objectif : Déterminer les chances de succès d'un atterrissage en fonction des caractéristiques de la mission (masse, orbite, site de lancement, etc.).

🚀 SpaceX Landing Prediction - Capstone Project
📝 Présentation du Projet
Ce projet final (Capstone) analyse les données de lancement de SpaceX pour prédire le succès de l'atterrissage du premier étage des fusées Falcon 9. La capacité de réutiliser ces boosters est le facteur clé qui permet à SpaceX de réduire drastiquement les coûts de mise en orbite.

Objectif : Déterminer les chances de succès d'un atterrissage en fonction des caractéristiques de la mission (masse, orbite, site de lancement, etc.).

📊 Performance du Modèle
Après une phase d'optimisation par GridSearchCV, le modèle final a atteint les performances suivantes :

Meilleur Score : 77.62% de précision.

Modèle Optimal : K-Nearest Neighbors (KNN).

Hyperparamètres : n_neighbors: 7, weights: 'uniform'.

🛠️ Pipeline de Traitement des Données
Le projet utilise un Pipeline Scikit-Learn automatisé pour garantir la reproductibilité :

Données Numériques : Imputation des valeurs manquantes par la médiane et normalisation via StandardScaler.

Données Catégorielles : Imputation par constante et encodage via OneHotEncoder.

Algorithme : Classification robuste avec KNeighborsClassifier.
