
# 🚀 SpaceX-Insight : Prédiction d'Atterrissage
Ce projet utilise le Machine Learning pour prédire le succès de l'atterrissage des boosters Falcon 9, permettant ainsi d'estimer les économies de coûts de lancement.

## 📊 Performance Finale
* Précision : 77.62%.
  
* Modèle : KNN (K-Nearest Neighbors).

* Optimisation : GridSearchCV avec 7 voisins.

##🛠️ Outils & Tech
📈 Flux de Travail
* Collecte : Extraction via l'API SpaceX (pas de scraping).

* Traitement : Pipeline automatisé (StandardScaler, OneHotEncoder).

* EDA : Visualisations interactives avec Folium et Dash.

* Modélisation : Évaluation de plusieurs algorithmes et sélection du meilleur score.
